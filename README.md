# 实在智能前端团队-技术介绍项目

## Getting Started

Install dependencies,

```bash
$ npm i
```

Start the dev server,

```bash
$ npm start
```

Build site app,

```bash
$ npm run build
```

deploy site app,

```bash
$ npm run deploy
```
